# practice_Sharedpreferences


# Requirements
- Kotlin 1.5.0
- Gradle 4.2.0
- Android min SDK 26

- etc

# Installation

- Manifest
If you want to use it as a global variable
Creates a class that inherits an Application and declares it.
```
<application
    …
    android:name = "com.shong.practice_sharedpreferences.MyApp">
    …
</application>

```
